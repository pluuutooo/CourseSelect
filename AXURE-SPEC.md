# ECNU 选课系统 Axure 原型规格文档

> 基于前端 JS/CSS 代码逆向分析，覆盖全部页面和交互。
> 配合 mock/ 目录下的 JSON 数据使用。

---

## 一、全局规范

### 配色
- 主色：#409EFF（Element UI 蓝）
- 成功：#67C23A（绿）
- 警告：#E6A23C（橙）
- 危险/已满：#FF6B6B / #FB5A5B（红）
- 信息灰：#909399
- 背景：#F5F7FA
- 白色卡片：#FFFFFF

### 字体
- 正文：14px，颜色 #303133
- 次要信息：12px，颜色 #909399
- 标题：16-18px，加粗

### 通用组件（Element UI 风格）
- 按钮：圆角4px，主色填充/白底蓝边两种
- 输入框：高36px，圆角4px，灰色边框
- 下拉选择器：同输入框样式
- 表格：白底，表头 #F5F7FA，行高48px，hover 高亮 #F5F7FA
- 标签页（Tabs）：下划线指示器，主色
- 弹窗（Dialog）：白色，圆角4px，半透明遮罩
- 进度条：高6px，圆角3px

## 二、页面结构（共5个页面）

```
Page 1: 登录页 (/login)
Page 2: 学生信息页 (/student)
Page 3: 选课轮次页 (/course-select/turns)
Page 4: 选课主页 (/course-select/:studentId/turn/:turnId/select)  ← 核心
Page 5: 未开放页 (/noturn)
```

---

## Page 1: 登录页

### 布局
- 居中卡片（宽400px），白色背景，圆角8px，阴影
- 标题：「选课」（大字居中）
- 副标题：「账号登录」

### 组件
1. 用户名输入框（placeholder: "用户名"）
2. 密码输入框（type: password）
3. 验证码输入框 + 验证码图片（右侧，可点击刷新）
4. 登录按钮（蓝色，宽100%）

### 交互
- 点击「登录」→ 跳转 Page 2
- 点击验证码图片 → 刷新验证码（换一张图片）
- 输入框为空时点登录 → 红色提示"必填"

## 四、交互流程图（在 Axure 中用 Flow 实现）

### 选课流程
```
[点击选课按钮]
     │
     ├── 频率检查（60秒内超10次？）
     │     └── 是 → 弹窗7（频率限制）
     │
     ▼ 否
[弹出结果弹窗 - loading状态]
     │
     ▼
[模拟校验结果]  ← 用 Axure 的 Case 切换
     │
     ├── 通过 → 显示"选课成功" → 关闭后刷新表格
     ├── 时间冲突 → 弹窗4（免听确认）
     │         ├── 免听 → 继续选课 → 成功
     │         └── 不免听 → 关闭
     ├── 人数已满 → 显示"该教学班已满"
     └── 学分超限 → 显示"选课学分已达上限"
```

### 退课流程
```
[点击退课按钮]
     │
     ├── 是打包课程？
     │     └── 是 → 弹窗3（打包确认）
     │
     ▼
[弹窗2 - 确认退课?]
     │
     ├── 取消 → 关闭
     └── 确定 → 弹出结果弹窗(loading)
              → 显示"退课成功"
              → 关闭后从表格移除该课程
```

## 五、Axure 搭建建议

### 页面组织
1. 在 Axure 中创建 5 个 Page
2. Page 4（选课主页）最复杂，建议：
   - 用 **Dynamic Panel** 做 Tab 切换（8个 State）
   - 用 **Repeater** 做课程表格（数据直接从 mock JSON 手动填入）
   - 弹窗用 **Dynamic Panel**（默认隐藏，点击时显示）

### 组件库
- 推荐使用 Element UI Axure 组件库（网上搜"Element UI Axure Library"）
- 包含按钮、表格、标签页、弹窗等现成组件

### Repeater 数据列（课程表格）
建议设置以下列：
```
id | code | courseName | credits | lessonName | teacher | timePlace | campus | stdCount | limitCount | status | pinned
```
将 `mock/query-lesson.json` 中的数据手动录入 Repeater。

### 交互变量
建议设置以下全局变量：
- `selectedCredits`（已选学分，初始11）
- `selectedCount`（已选门数，初始3）
- `creditLimit`（学分上限，30）
- `countLimit`（门数上限，10）

选课成功时：`selectedCredits += 课程学分`，`selectedCount += 1`
退课成功时：`selectedCredits -= 课程学分`，`selectedCount -= 1`
